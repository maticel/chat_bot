## General Info

Chat bot will be implemented using Tensorflow's Sequence to Sequence
model trained on 
<a href="https://www.cs.cornell.edu/~cristian/Cornell_Movie-Dialogs_Corpus.html">Cornell Movie Dialogue dataset</a>.

## Technologies
* Python 3.8.5
* Pandas 1.2

## Setup
To install all required packages use following commands:
~~~
cd ../chat_bot/
pip install -r requirements.txt
~~~

To run this application use following commands:
~~~
cd ../chat_bot/
python app.py
~~~
